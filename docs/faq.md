# FAQ

**В: Как добавить новую страницу?**  
О: Создайте новый `.md` файл в папке `docs/` (или подпапках). Добавьте front matter с `title`, `nav_order` и, при необходимости, `parent`.

**В: Как сделать раздел с подразделами?**  
О: Создайте страницу-раздел с `has_children: true`, а у дочерних страниц укажите `parent: <Название раздела>`.
